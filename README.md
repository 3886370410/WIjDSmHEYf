# 医院管理系统 java23

## 项目概述

医院管理系统是一款基于Web的医院信息管理应用，采用前后端分离的技术架构，前端使用Vue.js框架，后端采用Java和SpringBoot框架，实现了患者、医生和管理员三种角色的功能需求。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**：
  - IDEA
  - JDK 1.8
  - Maven 3.6.1
  - Mysql 8.0
  - Node.js 14
  - Navicat 17
  - Redis 5.0

## 功能模块

### 患者模块

- 预约挂号
- 我的挂号
- 报告单导出
- 住院信息
- 患者个人信息

### 医生模块

- 病患诊断
- 历史挂号列表
- 住院申请管理
- 医生个人信息

### 管理员模块

- 医生信息管理
- 患者信息管理
- 挂号信息管理
- 药物信息管理
- 检查项目管理
- 病床信息管理
- 排班信息管理

## 系统角色

系统包括患者、医生和管理员三个角色。

## 运行环境

- **服务器**: 推荐配置64位操作系统，至少4GB内存，建议使用Linux或Windows Server
- **数据库**: Mysql 8.0及以上版本
- **缓存**: Redis 5.0

```
后端目录结构

```
